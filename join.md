# 加入我们

TATEN 面向所有对计算机科学感兴趣的中学生开放。

我们看重的不是水平，而是热情、好奇心，以及持续学习的能力。

## 岗位需求

我们目前正在寻找对以下方向感兴趣的成员，一起参与项目与技术交流。

### 后端开发

你可以做什么：

1. 参与 TATEN-OJ 的后端开发；
2. 编写后端服务。

我们希望你：

1. 掌握至少一门后端语言（Golang/Java/etc.）；
2. 熟练开发与调试；
3. 能主动参与工作。

### 前端开发

你可以做什么：

1. 参与网站开发（如团队官网、工具站）
2. 构建 UI / 页面交互
3. 优化用户体验

我们希望你：

1. 掌握 Vue / React；
2. 有项目经验；
3. 对设计或交互有兴趣

### 系统开发

你可以做什么：

1. 交流未来的 TATEN-OS（暂定）的事宜。

我们希望你：

1. 能使用 C 语言编写简单的内核或者驱动；
2. 至少熟练掌握《操作系统导论》的相关内容；
3. 有技术热情。

### 设计 / 内容创作

你可以做什么：

1. 设计团队视觉（Logo / UI）；
2. 撰写技术文章或教程；
3. 制作演示 / 视频内容。

我们希望你：

1. 有一定审美或表达能力；
2. 愿意参与内容建设；
3. 会使用工具（如 Figma / PS / Markdown / LaTex 等）。

## 申请流程

1. 提交申请：点击下方按钮填写申请表单或提交 GitHub Issue。
2. 初步沟通：我们会邀请你进行一次简单的线上技术/意向交流。
3. 正式入队：获得团队权限，开启你的 TATEN 之旅！

如何申请

目前我们支持以下两种方式（任选其一）：

- [在 GitHub issus 提交申请 (推荐)](https://github.com/TATENcn/.github/issues/new?template=01_join_taten.yml)
- [在腾讯问卷填写申请问卷](https://wj.qq.com/s2/25766356/765a/)
